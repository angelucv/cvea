Aplicativos interactivos CVEA Suite

Los cinco demos (Bank, Insurance, Retail, Health, Control) se ejecutan en la aplicación siguiente. Use el menú lateral dentro del aplicativo para cambiar de módulo.

Si el aplicativo no carga: puede no estar desplegado en línea. Para ejecutarlo en su equipo, clone el repositorio del sitio, abra una terminal en la carpeta apps/cvea-suite-demos y ejecute: pip install -r requirements.txt y luego streamlit run Home.py.
