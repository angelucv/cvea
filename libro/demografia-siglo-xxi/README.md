# Libro: Demografía, Mortalidad y Previsión Social en la Venezuela del Siglo XXI — Un Enfoque Actuarial

Este libro fue compilado con la **plantilla CVEA Book** (basada en *Applied Machine Learning Using mlr3 in R*) que está en `Referencias/Plantilla_CVEA_Book_Applied Machine Learning Using mlr3 in R/`.

## Cómo publicar el libro en la web

Coloca en **esta carpeta** los archivos del libro ya compilado:

1. **Versión web (HTML):**  
   - El archivo principal `index.html` del build de Quarto.  
   - Y todas las carpetas/archivos que genere el build (p. ej. `index_files/`, recursos, etc.) para que los enlaces y estilos funcionen.

2. **Versión PDF:**  
   - Un solo archivo PDF del libro, por ejemplo:  
     `Demografia-Mortalidad-Prevision-Social-Venezuela-Siglo-XXI.pdf`  
   (o el nombre que tenga tu compilado).

La página [Libros y Materiales Docentes](../index.qmd) enlazará automáticamente:
- **Ver en línea** → `libro/demografia-siglo-xxi/index.html`
- **Descargar PDF** → el PDF que coloques aquí.

Si el compilado está en Referencias, copia solo la salida HTML (carpeta completa del build) y el PDF a esta carpeta.
